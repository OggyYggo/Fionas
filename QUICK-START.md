# ⚡ Quick Start Guide

**Get your Bohol Travel Agency website running in 5 minutes!**

## 📥 Step 1: Install & Setup (2 min)

```bash
# Install dependencies
npm install

# Build TypeScript
npm run build

# Start server
npm start
```

**You'll see:**
```
╔════════════════════════════════════════════╗
║  🌴 Bohol Travel Agency Server Running   ║
╠════════════════════════════════════════════╣
║  Server: http://localhost:3000
║  Status: Ready to accept bookings ✅
╚════════════════════════════════════════════╝
```

---

## 🌐 Step 2: Open Website (1 min)

1. Open browser
2. Visit: **http://localhost:3000**
3. You should see the website! 🎉

---

## 🧪 Step 3: Test Booking (2 min)

1. **Click "Tourist Spots"** section - See 6 destinations
2. **Scroll to "Book Now"** section
3. **Fill the form**:
   - Name: Your name
   - Email: your@email.com
   - Phone: +63 912 345 6789
   - Spot: Choose one
   - Date: Pick a future date
   - Guests: 2
   - Package: Premium
   - Click **"Book Now"**

4. **You'll see**: ✅ Success message with booking ID!

4. Refresh browser to see changes

2. Deploy to hosting (Vercel, Heroku, AWS)
3. Setup Google My Business
4. Submit to Google Search Console

## 📊 View Bookings

Option 1: **Browser Console**
- Open DevTools (F12)
- Go to Console tab
- Bookings appear when submitted

Option 2: **View JSON File**
```
logs/bookings.json
```
(Contains all booking records)

Option 3: **API Call**
```bash
curl http://localhost:3000/api/bookings
```

## 🐛 Troubleshooting

**Issue: "Port already in use"**
```bash
# Use different port
PORT=3001 npm start
```

**Issue: "Cannot find module"**
```bash
npm install
npm run build
```

**Issue: "Page not loading"**
- Wait 5 seconds
- Refresh browser (Ctrl+R)
- Check terminal for errors
